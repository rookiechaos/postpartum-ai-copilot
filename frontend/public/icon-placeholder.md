# Icon Placeholder

To complete PWA setup, you need to add app icons:

1. Create `icon-192x192.png` (192x192 pixels)
2. Create `icon-512x512.png` (512x512 pixels)

You can use any image editor or online tools to create these icons.

Recommended design:
- Simple, recognizable icon
- Background: #667eea (purple gradient)
- Icon: 🤱 or a simple baby/mother symbol
- Rounded corners (optional)

Place both files in the `public` directory.
