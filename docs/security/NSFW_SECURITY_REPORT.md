### 1. NSFW检测服务 ✅

**文件**: `backend/services/nsfw_detector.py`

**功能**:
- OpenAI Moderation API集成（主要检测方式）

- `SAFE`: 内容安全
- `SUSPICIOUS`: 内容可疑

## 检测流程

### 用户输入检测流程

```
用户输入
  ↓
NSFW检测器检查
  ↓
是否安全？
  ├─ 是 → 继续处理
  └─ 否 → 阻止请求 + 返回友好错误消息
```

### AI响应检测流程

```
AI生成响应
  ↓
NSFW检测器检查
  ↓
是否安全？
  ├─ 是 → 返回响应
  └─ 否 → 替换为安全默认响应 + 记录日志
```

## 配置选项

### 环境变量

```env
# 启用/禁用NSFW检测
NSFW_DETECTION_ENABLED=true  # 默认: true（推荐）

# 严格模式
NSFW_DETECTION_STRICT=true   # 默认: true（推荐）
                              # true: 阻止所有被标记的内容
                              # false: 只阻止明确不安全的内容
```

## 安全保证

### 多层防护

1. **第一层**: 用户输入检测
   - 在内容进入系统前检测
   - 阻止不当输入

2. **第二层**: AI响应检测
   - 在AI生成内容后检测
   - 确保输出安全

3. **第三层**: 医疗安全规则
   - 医疗建议验证
   - 危险信号检测

### 自动降级

- 如果OpenAI Moderation API不可用，自动使用关键词检测
- 确保检测服务始终可用
- 不会因为检测服务故障而暴露不当内容

### 日志和审计

- 所有被阻止的内容都记录在日志中
- 包含检测级别、原因、用户ID等信息
- 支持安全审计和问题排查
